# Android-template

Practica consumiento la api pública de https://www.themealdb.com/api.php.


El proyecto esta construido utilizando:

✓ MVVM

✓ Patrón repository

✓ Patrón builder (Para algunos modelos.)

✓ Retrofit client

✓ Jetpack Compose utilizando uiStates.

✓ Flows y corrutinas.

✓ Navigation

✓ Dagger Hilt
